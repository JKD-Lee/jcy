## 漂亮的Hugo主题-stack

## [stack主题页](https://themes.gohugo.io/hugo-theme-stack/)


- 文章图片自带长宽比信息
- 图片懒加载 (loading="lazy")
- 可以给分类 / 标签添加图片和介绍
- 归档页面
- 暗色模式 (跟随系统设置)
- 无 jQuery，纯原生 JavaScript
- 无 CSS 框架
